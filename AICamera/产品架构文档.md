# AI相机 - 产品功能架构文档

## 产品概述

AI相机是一款对标DOKA相机的手机端原生AI辅助缩放拍摄与构图应用，主打「AI自动焦段缩放 + AR实时构图引导」核心能力，帮助普通用户零门槛拍出高质量照片。

## 核心定位

### 极简纯粹原则
- 无信息流、无广告、无社区商城
- 启动直接进入全屏取景界面
- 全程专注拍摄本身

### 零门槛操作
- 无需手动调节快门、ISO、曝光等参数
- AI全自动适配拍摄环境
- 核心操作仅需「点击AI构图 + 按下快门」两步

## 核心功能模块

### 1. AI智能构图引擎

#### 场景识别
支持7类核心场景自动识别：

| 场景类型 | 焦段推荐 | 构图规则 |
|---------|---------|---------|
| 人像    | 中长焦(1.5-2x) | 三分法/黄金分割 |
| 风景    | 超广角(0.5x) | 三分法 |
| 美食    | 主摄(1x) | 中心构图 |
| 建筑    | 超广角(0.5x) | 对称/引导线 |
| 街景    | 主摄(1x) | 引导线 |
| 夜景    | 主摄(1x) | 中心构图 |
| 宠物    | 长焦(2x) | 黄金分割 |

#### 主体检测
- 实时目标检测（Object Detection）
- 多主体场景支持
- 主体追踪功能（动态场景）
- 置信度过滤（阈值：0.6）

#### 构图分析
自动生成构图建议：
- 最优取景框计算
- 构图网格线生成
- 对齐进度评估
- 构图提示生成

### 2. AR实时构图引导

#### 取景框系统
- 半透式AR推荐取景框
- 动态颜色变化（基于对齐进度）
  - 未对齐：白色半透明
  - 部分对齐：黄色/青色
  - 完全对齐：青绿色高亮
- 角标记增强视觉引导
- 对齐进度条显示

#### 网格线系统
支持多种构图规则网格：
- 三分法网格（4线）
- 黄金分割网格（4线）
- 中心十字线
- 对称中线

#### 水平仪
- 实时倾斜角度检测
- 基于加速度传感器
- 倾斜偏差数字显示
- 水平状态颜色反馈

### 3. 智能焦段缩放

#### 自动焦段匹配
基于场景和主体自动推荐：
- 人像：中长焦段优化背景虚化
- 风景/建筑：超广角呈现宏大场景
- 美食：主摄/微距展现细节
- 宠物：长焦抓拍动态主体

#### 平滑缩放动画
- 分段式渐进缩放（20步）
- 缩放时长：300ms
- 无卡顿、跳变感

### 4. 胶片滤镜系统

内置6款经典胶片滤镜：

| 滤镜名称 | 特点 | 推荐场景 |
|---------|------|---------|
| 柯达金    | 温暖经典，色彩饱满 | 人像/风景/街景 |
| 柯达Portra | 柔和自然，人像首选 | 人像 |
| 富士400H   | 清新淡雅，日系风格 | 人像/风景 |
| 爱克发Vista | 复古胶片质感 | 街景/建筑 |
| 伊尔福HP5  | 经典黑白，艺术感强 | 建筑/街景 |
| CineStill  | 电影感色彩 | 夜景/街景 |

滤镜参数：
- 温暖度（-1 ~ 1）
- 饱和度（-1 ~ 1）
- 对比度（-1 ~ 1）
- 颗粒感
- 衰减度
- RGB通道偏移

## 交互设计

### 主界面布局

```
┌────────────────────────────────┐
│  1x焦段    场景类型    [网格][水平仪] │ ← 顶部状态栏
├────────────────────────────────┤
│                                │
│         相机预览画面             │
│     + AR构图引导覆盖层           │
│                                │
├────────────────────────────────┤
│    [滤镜]  [快门]  [AI]  [相册]   │ ← 底部控制栏
└────────────────────────────────┘
```

### 操作流程

**基本拍摄流程：**
1. 启动应用 → 直接进入取景界面
2. 点击AI构图按钮 → 启用AI分析
3. AI自动识别场景和主体
4. 显示AR取景框和构图引导
5. 对齐取景框 → 触发震动反馈
6. 按下快门 → 完成拍摄

**高级功能：**
- 长按取景框 → 手动指定拍摄主体
- 左右滑动 → 切换滤镜
- 点击网格/水平仪按钮 → 开关辅助线

## 性能要求

### 流畅性标准
- AI实时检测帧率 ≥ 30fps
- 构图计算延迟 < 100ms
- 取景画面无拖影、卡顿

### 轻量化要求
- 安装包体积 ≤ 200MB
- 运行内存占用低
- 适配近3年主流中低端机型

### 功耗控制
- 连续拍摄30分钟无明显发烫
- 耗电量与系统自带相机持平

### 优化策略
- 端侧AI优先（ML Kit本地推理）
- 图像分析采样率优化（1280x720）
- 独立线程执行AI任务
- ProGuard代码优化和混淆
